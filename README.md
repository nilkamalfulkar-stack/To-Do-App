# To-Do App 📝

A minimalist and functional To-Do application built with **Flutter and Dart**. This app helps users manage daily tasks with a clean UI and smooth interactions.

## ✨ Features
* **Add Tasks:** Quickly add new items via a customized dialog box.
* **Delete Tasks:** Swipe-to-dismiss gesture with a red delete action for quick cleanup.
* **Checklist System:** Easily mark tasks as completed with a custom checkbox.
* **Persistent Storage:** (If you used Hive or SharedPreferences) Saves your tasks even after closing the app.

## 🚀 Tech Stack
* **Framework:** Flutter
* **Language:** Dart
* **State Management:** Provider / SetState (mention which one you used)
* **Database:** Hive (or mention if it's local memory)
